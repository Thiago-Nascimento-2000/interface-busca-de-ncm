<h3>Tecnologias: Next, React ts</h3>

<h3>Qual a Finalidade Dessa Criação ?</h3>
<p>Construido apartir de uma nescessidade que toda a minha equipe tinhamos para fim de minimizar a analise manual de NCM Invalido no banco de dados do sistema e posteriormente a consulta do ncm no site da siscomex para saber do status de validade, que demanda muito tempo para analise e resolução desse tipo de caso que ocorre recorrentemente.</p>
